## About

Multi-User Dungeon text adventure game.

## Features

- Text-based exploration
- Commands
- Inventory
- Combat

## Installation

### From App Store
Open the **App Store** in zOS and search for "MUD".

### Via npm
```bash
npm install @zos-apps/mud
```

## Usage

- Type commands to interact
- Use look to see surroundings
- Type help for commands



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
